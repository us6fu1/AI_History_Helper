# ИИ-помощник учителя

Код приложения находится в этой папке. Для обычного запуска из исходников:

```powershell
pip install -r requirements.txt
python main.py
```

Для сборки Windows-версии удобнее запускать `build_exe.bat` из корня проекта. Сборка использует `ИИ-помощник учителя.spec`, включает `assets`, `data` и папку `../Materials`.

Модели GGUF скачиваются отдельно и выбираются в интерфейсе приложения.
